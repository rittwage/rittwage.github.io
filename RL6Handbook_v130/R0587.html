Routine $0587:   File transfer management


This routine reads and transfers the demanded file (incl. shadow files).

Parameters:
[$C2]: Start/working/end Track of file to read & transfer
[$D7]: Start sector ID of file to find & read

Local parameters:
[$B5-$B6]: bytes left to be transferred to C64-client (~length of file)
[$B7]: =0 if file transfer complete
       <>0 if (at least one more) shadow files to be transferred
[$23]: ID of sector to find & read
