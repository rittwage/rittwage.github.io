 $1800:   Data Port B (ATN, CLOCK, DATA, Device #)

   +----------+---------------------------------------------------+
   | Bit  7   |   ATN IN                                          |
   | Bits 6-5 |   Device address preset switches:                 |
   |          |     00 = #8, 01 = #9, 10 = #10, 11 = #11          |
   | Bit  4   |   ATN acknowledge OUT                             |
   | Bit  3   |   CLOCK OUT                                       |
   | Bit  2   |   CLOCK IN                                        |
   | Bit  1   |   DATA OUT                                        |
   | Bit  0   |   DATA IN                                         |
   +----------+---------------------------------------------------+
