 $1D:   Synchronization value (Track-to-Track / Track 36)


This value is set in the $0745 routine: [$1D]:= [$0150]. It's the
number of data bytes after Track 36 Key Sector Keys until a SYNC is
detected (wrong bit rate set).



RapidLok-Reference:

 LDA $1D    : $049B***
 LDA $1D    : $049D***
 STA $1D    : $0760
 SBC $1D    : $0765
