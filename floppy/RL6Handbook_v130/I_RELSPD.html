Pirates remaster with drive speed control
                 - Windows XP console logfile


Hardware: Windows XP PC connected to 1541 via XAP1541 parallel cable.
          The 1541 has a manual drive speed control.

I used "rpm1541.exe" below to set the drive speed for each nibwrite run
(nibwrite shows different speeds). Because of the parallel port hardware
addon, my 1541 drive has a 1541-II ROM. I used my supplied original
Pirates PAL G64 images, both with Track 36 fixed: Key Sector shortened to
250 bytes.

                           !! Warning !!
      rpm1541.exe requires to write a signature to Track 41 !
            Never use rpm1541.exe with original disks !
